🎯 What Campers Do

30+ Hands-On Activities! 🎉

From building robots to making ice cream — every activity is hands-on, real, and loads of fun!

🤖 Robotics 🔬 Science 🌉 Engineering 💻 Tech & Coding 🎨 Creative 🎉 Fun & Games 🎁 Take-Home!
🤖 Robotics

Robot Building & Driving

The heart of Robocamp — real robots, real code, real competition experience!

🟡
Elementary

FLL Robot Build

Build a FIRST LEGO League robot from scratch! Learn about motors, sensors, and mechanical design using LEGO Mindstorms parts.

💻
Elementary

FLL Robot Code

Program FLL robots using block-based coding. Complete missions on the FLL competition field and learn autonomous programming!

⚔️
ElementaryThursday!

FLL Battle Bots! ⚔️

The grand finale for elementary campers! Teams compete head-to-head with their FLL robots. Who will win?! 🏆

🔵
Middle School

FTC Robot Build

Construct a real FIRST Tech Challenge robot using REV hardware — the same parts used by high school competition teams!

☕
Middle School

FTC Code (Java)

Write real Java code for the FTC robot! TeleOp and Autonomous modes using the REV Control Hub. Real software engineering!

🌐
Middle School

FTC SIM (Virtual Robots)

Program and drive virtual FTC robots on the FTC SIM platform. Choose blocks or Java. Build custom worlds too!

🎮
Middle SchoolThursday!

FTC Game Day! 🎮

The middle school finale — teams compete on the full FTC competition field with the robots they built and coded all week!

🔴
Both

Drive FTC & FRC Robots

Take the controls of real competition robots! Kids drive FTC robots AND massive FRC robots (the big ones from TV!). 😮

🐝
Both

Bristle Bots

Build tiny vibrating robots from toothbrush heads and small motors. A fast, exciting intro to motors and basic circuitry!

🔬 Science

Science Experiments That WOW!

Real chemistry, physics, and biology — hands-on and messy (the best kind of science)!

🐘
Elementary

Elephant Toothpaste 🐘

Watch a HUGE foam explosion erupt from a cylinder! A dramatic chemistry reaction using hydrogen peroxide and a catalyst. So cool!

🌧️
Both

Rain Cloud in a Jar

Create a mini rain cloud inside a jar using water, shaving cream, and food coloring. A beautiful weather science demo!

🌊
Elementary

Oobleck 🌊

Solid or liquid? This non-Newtonian fluid from cornstarch and water is both! Punch it fast — it's solid. Touch it slowly — it flows!

🍋
Elementary

Lemon Battery

Generate real electricity from lemons! Learn how electrochemical reactions create voltage and power small devices.

🌋
Both

Lava Lamp

Create a colorful lava lamp effect using oil, water, and Alka-Seltzer. Watch blobs float while learning about density!

🍬
Elementary

Rock Candy 🍬

Grow real edible crystals using sugar science! Watch crystals form over several days — then taste the results!

🚀
Both

Pop Rockets / Seltzer Rockets

Launch rockets using pressure from fizzy tablets! Experiment with fuel amount and launch angles to reach the greatest heights! 🚀

☀️
Both

Solar Pizza Oven ☀️

Build an oven from a pizza box, foil, and plastic wrap. Use the sun to melt chocolate and marshmallows — s'mores from solar energy!

🌉 Engineering

Engineering Challenges

Design, build, test, improve — the engineering design process in real action!

🌉
Elementary

Build a Bridge

Using only cardstock, build the bridge that spans the greatest gap. Teams of 2–3 compete to engineer the strongest structure!

🥚
Both

Egg Drop 🥚

The classic! Design a protective container from limited materials. Drop it from height — did your egg survive? Real engineering!

✈️
Both

Paper Airplanes ✈️

Fold efficient flying machines and test aerodynamics! Build a paper launcher using rubber bands. Fly for distance, accuracy, and time!

🪙
Both

Penny Boat

Fold aluminum foil into a boat that holds the most pennies before sinking. Learn buoyancy and structural design!

🪃
Middle School

Marshmallow Catapult

Build a catapult and launch marshmallows! Adjust for distance and accuracy while learning about mechanical advantage and energy.

🎳
Middle School

Marble Run

Design and build a marble track with loops, drops, and obstacles. Learn about gravity, momentum, and energy transfer!

💻 Technology

Tech & Coding

Real tools used by real engineers — from circuits to CAD design to computer programming!

⚡
Both

Snap Circuits ⚡

Build real working circuits! Learn about voltage, current, resistance, and Ohm's Law. Create LED displays and logic gates.

🖨️
BothTake-Home!

TinkerCAD (3D Design)

Design a 3D model in TinkerCAD — a real CAD tool used by engineers. Your design gets 3D printed and sent home with you! 🎁

🔹
BothTake-Home!

Glowforge Laser Cutter

Learn how laser cutters work and design a custom pendant. Middle schoolers use Onshape CAD. Everyone takes their pendant home! 🎁

⛏️
Elementary

Hour of Code (Minecraft)

Learn programming through the Minecraft Hour of Code! Loops, conditionals, and logic — all through your favorite game!

🔧
Middle School

Hardware Presentation

Deep dive into real robotics hardware — motors, servos, sensors, control hubs, and wiring. How do competition robots actually work?

🎨 Creative

Creative Activities

Making and crafting — because the best engineers are creative thinkers too!

🧶
Middle School

Paracord Bracelets

Learn weaving techniques to create a durable paracord bracelet. A mindful hands-on craft that doubles as a keepsake!

🏠
Elementary

Popsicle Stick Buildings

Build creative structures using popsicle sticks — towers, bridges, houses! A fun engineering challenge for check-out time.

🤝
Elementary

FLL Core Values Activities

Team-building inspired by FIRST's Core Values: Blanket Flip, Human Knot, and Watch Your Step (a blindfolded minefield game — builds trust!).

🎉 Fun

Fun, Games & Extras!

Because the best learning happens when you're having the most fun! 🎊

⚽
Daily

Dodgeball!

Every afternoon from 2:15–3:00 pm — a high-energy dodgeball session to reset and recharge after all that engineering!

🍦
BothThursday

Homemade Ice Cream! 🍦

Science + food = best ever! Put ice cream mix in a bag with ice and salt, shake for 10 minutes, eat the result. Toppings included!

💦
BothThursday

Splash Day! 💦

The camp finale! Outdoor water activities — Human Bubble is a fan favorite! Remind your camper to pack swim clothes on Thursday. 🏊

🎬
Daily

Snack & Movie Time

Daily 3:00–3:30 pm wind-down with snacks and a movie clip, robotics presentation, or explanation of real FTC/FRC competitions!
